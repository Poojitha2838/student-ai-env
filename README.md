# Student AI Environment

## Description
This project simulates a student learning environment where an AI agent takes actions like study, practice, or skip to maximize marks.

## Features
- step(), reset(), and state() functions
- Reward-based learning system
- Multiple difficulty levels (Easy, Medium, Hard)

## Actions
- Study: +10 marks
- Practice: +15 marks
- Skip: -5 marks

## Project Structure
- student_env.ipynb → Main environment logic
- inference.py → Baseline agent script
- openenv.yaml → Environment configuration

## Goal
To maximize student performance using intelligent actions.

## Note
This is a simplified OpenEnv-style environment demonstrating core concepts of reinforcement learning.
